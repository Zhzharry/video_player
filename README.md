# 基于多媒体播放器的图像灰度变换与增强演示系统

这是一个面向数字图像处理课程实验的桌面演示软件。软件保留视频、音频播放能力作为素材浏览入口，图片模式作为实验核心，用于展示灰度化、灰度级变换、二值化、Gamma 变换和直方图均衡化等效果。

## 功能

- 打开本地图片、视频、音频文件
- 图片原图/处理图左右对比
- 灰度化：`Gray = 0.299R + 0.587G + 0.114B`
- 灰度反转、亮度调整、对比度调整、阈值二值化、Gamma 变换、直方图均衡化
- 亮度、对比度、阈值、Gamma 参数实时调节
- 原图和处理图灰度直方图对比，二值化模式显示阈值辅助线
- 视频当前帧截图为图片，并自动进入灰度实验模式
- 处理结果按算法后缀另存为 png

视频由连续图像帧组成，因此可以把当前视频帧截取为静态图片，再对单帧应用灰度变换和增强算法。

## 安装

```bash
python3 -m pip install -r requirements.txt
```

## 运行

```bash
python3 main.py
```

## 图片实验流程

1. 点击“打开”选择图片，或把图片拖入窗口。
2. 在右侧“灰度变换实验”中选择算法。
3. 调节亮度、对比度、阈值或 Gamma 参数。
4. 观察左侧原图、中间处理图和右侧直方图变化。
5. 点击“还原原图”清除处理结果。
6. 点击“保存处理结果”另存图片。

默认保存命名：

- 灰度化：`{原名}_gray.png`
- 反转：`{原名}_invert.png`
- 亮度：`{原名}_bright.png`
- 对比度：`{原名}_contrast.png`
- 二值化：`{原名}_binary.png`
- Gamma：`{原名}_gamma.png`
- 均衡化：`{原名}_enhanced.png`

## 视频帧截图

打开视频后播放或暂停到目标画面，点击顶部“视频截图”。软件会保存：

```text
{视频名}_frame_{HH-MM-SS_mmm}.png
```

截图保存后会自动打开到图片实验区。

## 生成实验结果截图

把演示图片放入 `demo_assets/input/`，然后运行：

```bash
python3 demo_generate_results.py
```

生成的每种算法对比图会输出到 `demo_assets/output/`。

## 快捷键

- `O`：打开文件
- `Space`：播放/暂停
- `←`：后退 5 秒
- `→`：前进 5 秒
- `Ctrl+G`：跳转到指定时间
- `Ctrl+Z` / `Ctrl+Y`：图片撤销/重做
- `Ctrl++` / `Ctrl+-` / `Ctrl+0`：图片缩放
