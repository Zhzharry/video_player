# 图像灰度变换与增强实验报告

## 实验题目

基于多媒体播放器的图像灰度变换与增强演示系统。

## 实验目的

1. 理解彩色图像灰度化的加权平均方法。
2. 掌握常见灰度级变换对图像亮度、对比度和细节表现的影响。
3. 观察阈值二值化在前景/背景分割中的效果。
4. 通过直方图理解灰度分布和直方图均衡化的增强原理。
5. 认识视频由连续图像帧组成，单帧可以作为静态图像进行处理。

## 实验原理

彩色图像转灰度采用加权平均法：

```text
Gray = 0.299R + 0.587G + 0.114B
```

灰度反转用于观察亮暗关系反转：

```text
s = 255 - r
```

亮度调整通过整体平移灰度值改变图像明暗：

```text
s = r + b
```

对比度调整通过线性变换改变灰度差异：

```text
s = a * r + b
```

阈值二值化把图像分为黑白两类：

```text
s = 255 if r >= T else 0
```

Gamma 变换用于非线性灰度增强：

```text
s = 255 * (r / 255)^gamma
```

直方图均衡化根据累计分布函数重新映射灰度级，使灰度分布更均匀，从而增强灰度集中图像的对比度。

## 系统设计

系统由四个核心模块组成：

- 播放器模块：负责打开视频、音频和图片素材，保留播放、暂停、跳转、倍速和同目录切换能力。
- 图像处理模块：提供灰度化、反转、亮度、对比度、阈值、Gamma、直方图均衡化和直方图统计。
- 教学界面模块：显示原图/处理图对比、算法选择、参数调节、公式说明、图像尺寸和直方图对比。
- 文件保存模块：负责保存处理结果和视频截图，按算法或视频时间生成默认文件名。

## 操作流程

1. 运行 `python3 main.py`。
2. 打开图片素材。
3. 在右侧实验面板选择算法。
4. 调节算法参数，实时观察处理图和直方图变化。
5. 使用“还原原图”恢复初始状态。
6. 使用“保存处理结果”导出处理后的图片。
7. 打开视频时，可点击“视频截图”把当前帧保存为图片并进入实验模块。

## 结果分析

灰度化将彩色信息压缩到单通道灰度值，便于后续灰度级处理。灰度反转会交换亮区和暗区，使原本较暗的细节变亮。亮度调整会整体移动灰度分布，适合修正整体过暗或过亮的图像。对比度调整会拉大或压缩灰度差异，参数过大时可能出现饱和。阈值二值化能突出分割结果，但阈值选择直接影响细节保留。Gamma 小于 1 时暗部被提亮，Gamma 大于 1 时亮部被压暗。直方图均衡化能增强灰度集中图像的整体对比度，但对噪声较多的图像也可能增强噪声。

## 不足与改进

- 当前处理对象是静态图片或视频单帧，尚未实现实时视频滤镜。
- 当前算法覆盖基础灰度增强，后续可加入滤波、锐化、边缘检测等更多图像处理方法。
- 当前演示结果脚本使用单张输入图片，后续可扩展为批量素材报告生成。
- 直方图显示以教学观察为主，后续可增加坐标轴刻度和数值读数。
