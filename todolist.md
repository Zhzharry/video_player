## 一、实验主线：图像灰度变换与增强演示（最高优先级）

### 1. 软件定位

- [x] **确定实验题目**：基于多媒体播放器的图像灰度变换与增强演示系统
- [x] **明确软件用途**：打开图片素材，展示灰度化、灰度变换、二值化、直方图等数字图像处理效果
- [x] **保留播放器特色**：视频/音频播放作为素材浏览功能，图片模式作为实验核心功能
- [x] **补充实验说明入口**：在 README 或文档中说明本软件与数字图像课程中灰度变换内容的关系

### 2. 灰度处理基础功能

- [x] **彩色图像转灰度**：使用加权平均法 `Gray = 0.299R + 0.587G + 0.114B`
- [x] **灰度反转**：实现 `s = 255 - r`，用于观察亮暗关系反转效果
- [x] **亮度调整**：实现 `s = r + b`，通过滑块调节整体明暗
- [x] **对比度调整**：实现 `s = a * r + b`，通过滑块调节灰度差异
- [x] **阈值二值化**：实现 `s = 255 if r >= T else 0`，通过阈值滑块观察分割效果
- [x] **Gamma 变换**：实现 `s = c * r^γ`，用于非线性灰度增强
- [x] **直方图均衡化**：增强灰度分布较集中的图像对比度

### 3. 教学演示界面

- [x] **原图/处理图对比显示**：支持左右对比或切换查看
- [x] **算法选择控件**：提供灰度化、反转、亮度、对比度、阈值、Gamma、直方图均衡化等选项
- [x] **参数调节控件**：为亮度、对比度、阈值、Gamma 提供实时滑块
- [x] **公式显示区域**：选择不同算法时显示对应灰度变换公式
- [x] **处理结果说明**：显示当前算法名称、参数值和图像尺寸
- [x] **一键还原原图**：清除所有处理结果，恢复原始图像
- [x] **保存处理结果**：另存为 `_gray`、`_binary`、`_enhanced` 等后缀图片

### 4. 灰度直方图模块

- [x] **显示原图灰度直方图**：统计 0-255 灰度级出现次数
- [x] **显示处理后直方图**：用于观察灰度变换前后分布变化
- [x] **直方图对比**：原图和处理图直方图可上下或左右显示
- [x] **阈值辅助线**：二值化模式下在直方图上标出当前阈值位置
- [x] **均衡化效果展示**：对比均衡化前后的灰度分布变化

### 5. 视频素材结合方式

- [x] **视频截图转图片处理**：从当前视频帧截取图片，进入灰度实验模块

- [x] **截图后自动打开**：截取当前帧后直接显示在图片处理区
- [x] **文档中说明视频帧本质**：视频由连续图像帧组成，因此可以对单帧应用灰度变换
